# 09. Automation

> Agents running on schedules, without you asking.

## What this is

Automation is what makes an AI system run in the background: scheduled tasks, triggered workflows, and recurring jobs that produce value while you sleep.

A system without automation is just a smart chat window. A system *with* automation is a team member.

## What lives here

```
automation/
├── launchd/          macOS LaunchAgents for scheduled jobs
├── cron/             cross-platform cron jobs
├── webhooks/         event-triggered workflows
└── examples/         daily digest, weekly review, morning briefing
```

## Why it matters

The best parts of your AI system are the ones you never have to trigger. A daily digest arriving at 6 AM. A weekly review prompt every Sunday. A summary of your overnight messages waiting when you log in.
