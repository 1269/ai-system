# The 9 Components of an AI System

*Overview guide. Full video walkthrough coming soon at [youtube.com/@jashia](https://youtube.com/@jashia).*

## The definition

**An AI System is the curation of an Agent's components so that you can create better outputs.**

Agents (ChatGPT, Claude Code, Gemini, the ones you build) are the *runtimes*. The AI System is the infrastructure they run on. Swap agents, keep the system.

## The 9 components, 3 tiers

### Tier 1 — Foundation

1. **Input & Capture** — how you talk to your agent
2. **Context Engineering** — your agent's operating manual
3. **Memory** — your agent's notebook
4. **The Model** — your agent's brain

### Tier 2 — Capabilities

5. **Skills** — capabilities your agent can call
6. **Integrations** — tools your agent can reach
7. **Knowledge Base** — the library your agent can reference

### Tier 3 — Power Layer

8. **Orchestration** — multiple agents coordinating
9. **Automation** — agents running on schedules

## How to use this repo

Each numbered folder in the repo corresponds to one component. Each guide in this folder (01-capture.md through 09-automation.md) is a deep dive on that component.

If you're just starting out, read this overview, then work through the guides in order. If you already have a system and want to upgrade one piece, jump to the relevant guide.
